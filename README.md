# 传统建筑文化特征多模态识别与提取系统

基于 Vue 3 的前端演示系统，实现建筑图像的类型与风格智能识别与标注。**纯前端实现，无需后端与数据库。**

## 功能模块

### 模块一：多模态数据输入
- **图像输入**：拖拽/点击上传单张或多张建筑照片，支持航拍图、实拍图、历史档案照
- **视频输入**：扩展入口（演示用）
- **文本输入**：输入建筑描述，自动提取地域、年代、风格关键词

### 模块二：建筑检测与区域定位
- 自动检测建筑主体边界框
- 构件检测（屋顶、斗拱、檐口、门窗等）并高亮标注

### 模块三：建筑类型与风格识别（核心）
- **类型识别**：宫殿、宗教、民居、园林、防御建筑
- **风格识别**：地域风格（北方官式、江南水乡、闽南、藏式）、朝代风格（唐/宋/明/清）、民族风格
- **可解释化**：展示判定依据区域与文字说明

### 模块四：文化特征提取
- 结构特征：对称性、开间、屋顶坡度
- 装饰纹样：龙凤纹、莲花纹、云纹
- 材质色彩：主色调、社会等级、功能属性

### 模块五～九
- 多建筑对比分析
- 风格演化时间轴
- 建筑文化知识图谱
- 地图分布可视化
- 沉浸式全屏浏览

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- Vue 3 + Composition API
- Vue Router
- Pinia
- Element Plus
- ECharts
- Vite
