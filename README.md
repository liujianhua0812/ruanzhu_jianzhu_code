# 传统建筑文化特征多模态识别与提取系统

基于 Vue 3 的前端演示系统，实现建筑图像的类型与风格智能识别与标注。**纯前端实现，无后端与数据库。**

## 功能模块

### 模块一：多模态数据输入
- **图像输入**：拖拽/点击上传单张或多张建筑照片，支持航拍图、实拍图、历史档案照
- **视频输入**：扩展入口（演示用）
- **文本输入**：输入建筑描述，自动提取地域、年代、风格关键词

### 模块二：建筑检测与区域定位
- 自动检测建筑主体边界框
- 构件检测（屋顶、斗拱、檐口、门窗等）并高亮标注

### 模块三：建筑类型与风格识别（核心）
- **类型识别**：宫殿、宗教、民居、园林、防御建筑
- **风格识别**：地域风格（北方官式、江南水乡、闽南、藏式）、朝代风格（唐/宋/明/清）、民族风格
- **可解释化**：展示判定依据区域与文字说明

### 模块四：文化特征提取
- 结构特征：对称性、开间、屋顶坡度
- 装饰纹样：龙凤纹、莲花纹、云纹
- 材质色彩：主色调、社会等级、功能属性

### 模块五～九
- 多建筑对比分析
- 风格演化时间轴
- 建筑文化知识图谱
- 地图分布可视化
- 沉浸式全屏浏览

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 说明
核心功能，”建筑识别与标注“页的识别结果中，“建筑边界”以及“构件标注”两中属性的标注框均可手动拖拽位置，“构件标注”框还支持双击编辑左上角label以及双击“构件标注”框复制一个新框的功能

1.识别结果由前端模拟算法随机生成建筑解析文字，识别结果与图片内容的关联性需手动改写描述，用于截图演示
2.并不支持视频输入后识别的功能，只需在“图像输入”上传图片后，点击“视频输入”，右侧的识别结果标题会变化为“视频识别结果”由此方法模拟视频识别的结果
3.识别结果中的描述字段双击即可进行更改，目前是随机一些建筑描述，与图片未必相符，此处需手动更改描述（支持描述双击更改，失焦后自动保存，系统中所有功能页所展示的内容以及功能页中能编辑更改描述的的项详见“系统演示说明文档”）
